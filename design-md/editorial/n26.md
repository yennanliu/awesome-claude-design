# N26 — Editorial Minimalism (Light Fintech)

Reference DESIGN.md for a minimalist neobank: warm off-white canvas, one confident petrol-green accent, oversized bold display, and a single full-bleed lifestyle photo. Restraint and whitespace do the work — editorial minimalism applied to consumer banking. Reference captured on the EU homepage.

## 1. Visual Theme & Atmosphere

"The first bank you'll love." N26's whole point is that banking can feel calm and human, so the design is spare: an off-white stage, enormous confident type, one green, and a real photo of someone using the app. No clutter, no dashboards on the marketing page — just clarity and a little warmth.

Mood: calm, premium-approachable, confident, human. Less corporate-bank, more well-designed everyday object.

## 2. Color Palette & Roles

```
--bg:              #f7f5f0   /* warm off-white canvas */
--bg-alt:          #efece5   /* section lift */
--surface:         #ffffff   /* card */
--ink:             #16171a   /* near-black text, dark pills */
--text:            #16171a
--text-muted:      #5b5f63
--text-dim:        #8b8f93
--border:          #e4e0d8
--border-strong:   #d3cec4
--on-photo:        #ffffff   /* text over imagery */

--accent:          #1c7a67   /* N26 petrol green — CTA, links, active */
--accent-hover:    #156353
--accent-deep:     #0f4a3e
--accent-soft:     #dff0ea   /* mint tint for callouts, secondary buttons */

/* status (app / balances) */
--pos:             #1c7a67   /* credit / positive (shares brand green) */
--neg:             #d64545   /* debit / decline */
```

Rule: petrol green is the only branded hue and the single action colour; the mint tint softens callouts and secondary buttons. Imagery supplies any other colour. Never introduce a second saturated brand hue, and keep status red for money only.

## 3. Typography Rules

- **Display / hero:** a bold geometric-ish grotesque (`N26 Steilneue` → `Graphik`, `Inter`), weight 700, very tight tracking, **huge** ("The first bank you'll love").
- **Wordmark:** "N26" set with the signature overline — treat as a lockup.
- **Body / UI:** same family or `Inter`, weight 400/500, line-height 1.55, `--text-muted` for secondary.
- **Numerals:** tabular for balances, prices, plan fees, transaction lists.
- **Mono:** `JetBrains Mono`, `SF Mono` — IBANs, transaction IDs, OTPs only.

Scale: 12 / 14 / 16 / 18 / 22 / 28 / 40 / 56 / 80.

Hero display runs 56–80px; body 16–18px. Generous line-height and space around headings.

## 4. Component Stylings

**Photo hero** (the centrepiece)
- One full-bleed lifestyle photo (person + phone showing the app), warm and real.
- Centered oversized bold display in `--on-photo` white with a soft scrim for legibility; one rounded CTA below.

**Buttons**
- Primary: `--accent` fill, white text, fully rounded pill, weight 600 ("Open bank account"). On the nav, a dark `--ink` pill variant is also used.
- Secondary: `--accent-soft` mint fill, `--accent-deep` text, pill.
- On photo: white pill with `--ink` text.

**Cards / list items**
- White fill, 1px `--border`, radius 16px, generous padding, soft shadow.
- App/phone mockups feature real UI with green accents; keep chrome minimal.

**Plan / pricing**
- White cards, big tabular price + short benefit list with green checks; one plan highlighted with a 1px `--accent` border.

**Balance rows (app)**
- 56px rows, merchant icon + name, tabular amount right-aligned, +/− with sign and `--pos`/`--neg`.

**Nav**
- Off-white/white bar, overline "N26" wordmark left, sparse text links, a soft `--accent-soft` "Log in" pill + a solid `--accent` (or `--ink`) "Open bank account" pill, locale switcher.

**Inputs**
- `--surface` fill, 1px `--border`, radius 12px (large tap targets). Focus: 2px `--accent` ring.
- Amount/IBAN fields tabular / mono, right-aligned for currency.

## 5. Layout Principles

- 1120px max content shell, 24px gutter; the hero photo breaks full-bleed.
- Very generous vertical rhythm (64–112px between blocks) — whitespace is the brand.
- 4px base grid. Spacing scale: 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96.
- Alternating: full-bleed photo → centered narrative → clean 2–3-up feature cards.
- Mobile-first for the app; marketing responsive desktop-down.

## 6. Depth & Elevation

Light, airy — depth from imagery and soft shadow, not borders or heavy chrome.

- Card: `0 2px 12px rgba(20,20,20,0.05)`; hover `0 8px 24px rgba(20,20,20,0.08)`.
- Overlay on photo: gentle scrim `rgba(0,0,0,0.28)` behind text.
- No neumorphism, no gradients on flat surfaces (gradients only as photo scrims), no glow.

## 7. Do's and Don'ts

**Do**
- Lead with one calm full-bleed photo + huge confident display + a scrim.
- Keep petrol green as the single action colour; mint tint for secondary.
- Use enormous whitespace; let headings breathe.
- Tabular numerals for balances/prices; overline "N26" wordmark as a lockup.
- Fully rounded pill buttons.

**Don't**
- Add a second saturated brand hue or reuse status red decoratively.
- Crowd the page — restraint is the identity.
- Use dark mode as the default marketing canvas.
- Let white text sit on a photo without a scrim.
- Square off buttons or stack heavy shadows.

## 8. Responsive Behavior

- Hero photo crops to keep the subject in frame; display scales 80 → 32 on mobile.
- Nav collapses to a menu; the green "Open bank account" pill persists.
- Feature cards stack; keep the generous gaps.
- App surfaces mobile-first with 56px rows and bottom sheets.

## 9. Motion Notes

- Quiet, calm motion (200–300ms ease); gentle fades and rises.
- Honour `prefers-reduced-motion`. Stillness reads as premium — motion is punctuation.

## 10. Agent Prompt Guide

Bias: warm off-white canvas (#f7f5f0), single petrol-green accent (#1c7a67) + mint tint (#dff0ea), bold grotesque display (Graphik/Inter) at 56–80px, one full-bleed lifestyle photo hero with a scrim, overline "N26" wordmark, fully rounded pill buttons, tabular numerals for balances/prices, enormous whitespace, calm motion.

Reject: second saturated brand hue, status red as decoration, dark-mode marketing default, crowded layouts, unscrimmed white text on photos, squared buttons, heavy shadows, non-tabular amounts.
